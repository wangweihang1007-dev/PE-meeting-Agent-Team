# 核查与少样本风格

## 来源和冲突

先按字段记录来源口径，不静默套用一个全局排序：

| 字段 | 优先来源 |
|---|---|
| 成立时间、城市、股权、实控人 | 最新工商/企查查/天眼查/Choice中国企业库摘录、官方公示 |
| 融资事件 | 工商变更、投资方/公司公告、最新会议口径、BP |
| 本轮条款 | 最新会议纪要、融资材料、管理层明确陈述 |
| 产品与技术 | 最新会议/产品资料、BP、专利/认证 |
| 生产、客户、订单 | 最新会议纪要或逐字稿；公开信息只作辅助 |
| 财务 | 审计/报表、最新会议确认口径、BP预测 |
| 市场 | 有年份和口径的研报、行业协会、公司 BP |

企查查/工商资料只用于项目录入，不进入会议纪要修正、QA 或总结。

冲突处理：

- 工商登记 > 官方披露/公告/审计报表 > 口头陈述/BP。
- 同类来源中较新时点优先。
- 无法消解时，表格采用保守表述或留空；备注或交付说明保留全部冲突口径。
- 每个关键数字要能说明来自 BP、纪要、摘要、工商/QCC、财报、官网、公告或其他文件。
- P 列写来源组合，如 `资料来源：BP、纪要、工商`。有冲突时追加 `冲突：...`。

## 少样本揭示的做法

- 旧 BP 的拟融资方案不直接照搬；最终融资信息采用较新的会议、工商或交割事实。
- 最新股权采用企业查询的精确比例，不用会议中的概略比例覆盖。
- 发货、确认收入、订单、预算严格区分。
- 量产、送样、验证、接触中、在研、规划分开写。
- 主营业务压缩到具体产品，不写宽泛赛道。
- 客户按已贡献收入、开发/送样、接触中分层。

## 冲突处理例

- BP：2028 年科创板或创业板；较新会议：倾向创业板、计划 2028 年申报。  
  写法：`科创板或创业板；计划于2027-2028年上市`，或在证据更充分时写 `倾向创业板；计划2028年申报`。不得写成已确定上市。

- BP：A 轮拟融资 1 亿元；会议与工商：A/A+ 合计融资 2.4 亿元。  
  写法：已投机构列实际轮次和投资方；前轮估值列投前 5 亿元、投后 7.4 亿元。

- 会议：今年收入约 2 亿元；历史实际：去年确认 5857 万、发货约 8000 万。  
  写法：`2025年确认收入5857万元；2026年预计确认收入2.01亿元`，不能合并为“营收8000万元”。

## 表述限制

- 把 `全球领先、国内唯一、填补空白、行业第一` 视为公司主张，除非有独立证据。
- 把 `已合作、客户包括` 拆成 `已供货/已量产`、`送样/验证中`、`接触中`。
- 把 `预计、目标、计划、意向` 原样保留，不改写为事实。
- 市场数据必须保留年份、区域和口径；口径不明时不填精确值。
- 不根据注册资本反推估值，不根据投资方名单反推融资金额，不根据订单反推收入。

## 白描和低 bit 写法

- 去掉形容词、副词和评价词，只写事实。
- 能用缩略语就不用全称：BP、纪要、摘要、QCC、财报、工商、官网、公告。
- 句子优先短句、分号、数字。少用连接词。
- 写状态词：量产、送样、验证、接触、在研、规划、预计、确认收入、发货、订单。
- 禁用或降格：领先、唯一、首创、优质、资深、显著、快速、深度、强大、完善、重要、持续、全面、丰富。
- 好：`2025确认收入5857万；发货约8000万；2026预计确认收入2.01亿。`
- 避免：`公司收入增长较快，业务发展态势良好。`

## Excel 风格

- 以 `assets/项目表-录入参考指引.xlsx` 和相邻正式项目行为最高优先级。
- 新版表格有效列为 A:Q；P 为备注，Q 为否决原因；已取消旧版 `是否通过`。
- 表头蓝色、说明红色、正式录入黑色的既有风格不得被重做。
- 保持自动换行、顶端/居中对齐和超宽价值列。
- 新行复制上一条正式项目行的格式，不复制其值。
- 不新增工作表、图表、封面或说明页，除非用户明确要求。
- 完成后检查 A:Q 的值、格式和换行。
